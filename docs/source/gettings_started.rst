Getting started
===============


Installation
------------

You can install **Usernado** with ``pip`` or `poetry <https://python-poetry.org/>`_.

With pip
########

``(.venv) $ pip install usernado``

With poetry
###########

``(.venv) $ poetry add usernado``

From Github
###########

``(.venv) $ pip install git+https://github.com/reganto/usernado``
